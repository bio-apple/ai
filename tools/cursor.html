Cursor 教程 2026

Cursor 是基于 VS Code 的 AI 原生代码编辑器，支持 Tab 补全、内联编辑、Agent 自主编程和 MCP 扩展。

快速入门：下载安装 → 按 Cmd+I 打开 Agent → 用 @ 引用文件 → 在 .cursor/rules 定义项目规范。
  • Tab 补全 — 预测式多行代码补全
  • Cmd+K — 选中代码内联 AI 编辑
  • Agent 模式 — 自主搜索、编辑、运行终端
  • Plan 模式 — Shift+Tab 先制定计划再执行
