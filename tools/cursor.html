Cursor 教程 2026

基于 VS Code 的 AI 原生代码编辑器，将大语言模型深度集成到编辑体验的每一层。支持 Tab 补全、内联编辑、Agent 自主编程和 MCP 扩展。

快速入门

  1. 访问 cursor.com/download 下载安装包
  2. 首次启动时导入 VS Code 扩展和设置
  3. 打开项目文件夹：File → Open Folder
  4. 按 Cmd+I（Mac）或 Ctrl+I（Win）打开 Agent 面板
  5. 用 @ 引用文件，给 Agent 更多上下文
  6. 在 .cursor/rules/ 中创建项目规则文件

核心功能

  • Tab 补全 — 预测式 AI 自动补全，支持多行代码
  • Cmd+K — 选中代码后内联 AI 编辑
  • Agent 模式 — 自主搜索代码库、编辑文件、运行终端
  • Plan 模式 — Shift+Tab 先制定计划再执行
  • Rules — .cursor/rules/ 定义项目编码规范
  • MCP 服务器 — 连接数据库、GitHub 等外部系统
  • Cloud Agents — 浏览器/手机远程启动编程任务

Cmd+L / Ctrl+L打开 Chat 对话
