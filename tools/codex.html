OpenAI Codex 教程 2026

OpenAI 的编程代理系统，不再是早期的 API 模型，而是完整的自主编程平台。支持 CLI、IDE 扩展、桌面 App 和云端 GitHub 集成。

快速入门

  1. 确保有 ChatGPT 账号（Plus / Pro / Business 计划）
  2. 安装 CLI：npm install -g @openai/codex，运行 codex 登录
  3. 或下载桌面 App：访问 chatgpt.com/codex
  4. 在 VS Code / Cursor 中安装 OpenAI Codex 扩展
  5. 运行 /init 生成 AGENTS.md 项目配置文件
  6. 连接 GitHub，在云端分配任务并自动生成 PR

核心功能

  • 四大入口 — CLI、IDE 扩展、桌面 App、Codex Cloud
  • AGENTS.md — 项目级 AI 指令手册，定义编码规范
  • Worktrees — Git 工作树隔离，并行执行多个任务
  • Skills — 将重复工作流封装为可复用技能
  • MCP 集成 — 连接 Jira、Linear 等外部工具
  • PR 自动化 — 在 GitHub Issue/PR 中 @codex 自动处理
  • 沙箱模式 — 安全审批机制，控制 Agent 权限
