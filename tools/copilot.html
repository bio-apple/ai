Microsoft Copilot 教程 2026

微软 AI 产品家族，包含 GitHub Copilot（代码补全与 Agent）和 Microsoft 365 Copilot（Word、Excel、PPT、Outlook 办公助手）。深度整合 VS Code、Visual Studio 和 Microsoft 生态。

快速入门

  1. GitHub Copilot：访问 github.com/features/copilot 订阅（学生/开源维护者免费）
  2. 在 VS Code 安装「GitHub Copilot」和「GitHub Copilot Chat」扩展
  3. 登录 GitHub 账号，编码时 Tab 接受补全建议
  4. 按 Ctrl+I（Win）/ Cmd+I 打开 Copilot Chat 对话
  5. M365 Copilot：通过 Microsoft 365 企业订阅使用 Word/Excel/PPT AI 功能
  6. 通用助手：访问 copilot.microsoft.com

核心功能

  • 代码补全 — 行级/块级智能补全，支持 30+ 编程语言
  • Copilot Chat — IDE 内对话，解释代码、生成测试、修复 Bug
  • Copilot Agent — 自主执行多步编程任务（VS Code 预览）
  • Copilot Workspace — 从 Issue 到 PR 的端到端开发流程
  • M365 Copilot — Word 起草、Excel 分析、PPT 生成、Outlook 邮件
  • 企业合规 — 代码建议不用于模型训练（Business 计划）

⌨️ 快捷键速查（VS Code）

快捷键功能
Tab接受 Copilot 补全建议
Alt + ] / Alt + [切换下一个/上一个建议
Ctrl+I / Cmd+I打开 Copilot Chat 内联对话
Ctrl+Shift+I打开 Copilot Chat 侧边栏
