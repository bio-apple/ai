Kimi 教程 2026

月之暗面（Moonshot AI）出品的国产 AI 助手，以超长上下文窗口和强大的文件解析能力著称。支持 PDF、Word、Excel 等格式上传，适合处理合同、论文、报告等长文档。

快速入门

  1. 访问 kimi.moonshot.cn 或下载 Kimi App
  2. 手机号 / 微信登录，新用户有免费额度
  3. 点击「上传文件」导入 PDF、Word、PPT 等文档
  4. 开启「联网搜索」获取最新信息
  5. 开发者：访问 platform.moonshot.cn 获取 API Key

核心功能

  • 超长上下文 — 支持百万字级文档一次性分析
  • 文件解析 — PDF、Word、Excel、PPT 智能提取与问答
  • 联网搜索 — 实时检索互联网信息并整合回答
  • K2 模型 — 强化推理与代码能力，适合技术文档
  • 多轮对话 — 基于已上传文件持续追问
  • 开放平台 API — 兼容 OpenAI 格式，便于接入应用
