# 办公 Prompt 最佳实践（已验证）

## 1) 会议纪要与待办提取

**适用场景**：会议录音/文字整理。  
**输入变量**：`<会议内容>` `<参与角色>` `<项目目标>`

```text
请将以下会议内容整理为“可执行纪要”：
<会议内容>

要求：
1. 会议结论（按主题分组）
2. 待办事项（负责人、截止时间、依赖项）
3. 风险与阻塞
4. 需要上级决策的问题
```

**验证清单**
- 待办是否有责任人和时间
- 决策事项是否单列
- 风险是否清晰可追踪

---

## 2) 邮件草拟（多对象版本）

**适用场景**：跨部门沟通、客户同步。  
**输入变量**：`<背景>` `<目标对象>` `<目标动作>`

```text
请基于以下信息生成邮件：
- 背景：<背景>
- 对象：<目标对象>
- 目标动作：<目标动作>

输出三版：
1. 对管理层（结论先行）
2. 对执行团队（任务清单）
3. 对外部客户（友好专业）

每版包含：标题、正文、下一步动作、截止时间。
```

**验证清单**
- 是否按对象区分语气
- 是否包含明确 next action
- 是否避免歧义表述
