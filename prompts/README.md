# Prompt 最佳实践库

本目录用于沉淀**经过验证的高质量 Prompt**，作为项目长期复用的最佳实践资产。

## 目录结构

- `coding/`：编程开发
- `writing/`：写作表达
- `research/`：科研学习
- `data-analysis/`：数据分析
- `productivity/`：办公效率
- `marketing/`：市场分析
- `index.json`：统一索引（便于脚本/页面接入）

## 验证标准（入库门槛）

1. **可复现**：至少在 2 个模型上可稳定输出可用结果。
2. **可审阅**：输出包含结构化段落或清单，便于人工检查。
3. **可控**：明确输入变量、边界和失败回退策略。
4. **可落地**：结果可直接用于业务（文档、代码、分析结论、清单）。

## 模板规范

每条 Prompt 建议包含：

- 适用场景
- 输入变量
- Prompt 模板（可直接复制）
- 验证清单（结果质量检查）
- 失败回退（模型跑偏时如何补救）

## 维护建议

- 新增 Prompt 时同步更新 `index.json`
- 每月抽检：过时术语、失效链接、冗余约束
- 如发现低稳定性，标记 `status: draft` 并暂不推荐
