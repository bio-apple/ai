# 市场分析 Prompt 最佳实践（已验证）

## 1) 竞品对比（可落地版本）

**适用场景**：产品定位、方案选型、投放策略。  
**输入变量**：`<我方产品>` `<竞品列表>` `<目标市场>`

```text
请做竞品分析：
- 我方产品：<我方产品>
- 竞品：<竞品列表>
- 目标市场：<目标市场>

输出：
1. 对比维度表（功能、价格、渠道、品牌、增长）
2. 竞品优势/弱点
3. 我方机会点（按影响力排序）
4. 30天可执行动作清单
```

**验证清单**

- 对比维度是否完整
- 动作是否可执行
- 排序逻辑是否清晰

---

## 2) 用户画像与信息主张

**适用场景**：新市场切入、内容策略规划。  
**输入变量**：`<产品信息>` `<目标人群>` `<当前渠道>`

```text
请基于输入构建营销基础框架：
- 产品信息：<产品信息>
- 目标人群：<目标人群>
- 当前渠道：<当前渠道>

输出：
1. 3类核心用户画像（痛点/触发点/决策因素）
2. 每类人群的价值主张与卖点表达
3. 渠道与内容建议（短视频/图文/私域/搜索）
4. A/B 测试方案（指标、样本、周期）
```

**验证清单**

- 用户画像是否可区分
- 主张是否对应痛点
- 测试方案是否可测量
