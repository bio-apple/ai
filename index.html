C

Claude

Anthropic 出品的 AI 助手，包含网页版 Claude.ai 和终端编程工具 Claude Code。擅长复杂推理、长文档分析和高质量代码生成。

快速入门

  1. 访问 claude.ai 注册账号（支持 Google 登录）
  2. 下载桌面版 App（macOS / Windows），体验更快的响应速度
  3. 安装 Claude Code：curl -fsSL https://claude.ai/install.sh | bash
  4. 在项目目录运行 claude，按提示完成浏览器登录
  5. 创建 CLAUDE.md 文件，写入项目规范和编码约定

核心功能

  • Projects — 为每个项目创建独立工作区，上传知识库文件
  • Artifacts — 实时预览代码、文档和网页原型
  • Skills — 将重复工作流封装为可复用技能包
  • Connectors — 连接 Google Drive、Slack、Notion 等工具
  • Claude Code — 终端中自主读写文件、运行命令、提交代码
  • Extended Thinking — 深度思考模式，适合复杂问题

X

OpenAI Codex

OpenAI 的编程代理系统，不再是早期的 API 模型，而是完整的自主编程平台。支持 CLI、IDE 扩展、桌面 App 和云端 GitHub 集成。

快速入门

  1. 确保有 ChatGPT 账号（Plus / Pro / Business 计划）
  2. 安装 CLI：npm install -g @openai/codex，运行 codex 登录
  3. 或下载桌面 App：访问 chatgpt.com/codex
  4. 在 VS Code / Cursor 中安装 OpenAI Codex 扩展
  5. 运行 /init 生成 AGENTS.md 项目配置文件
  6. 连接 GitHub，在云端分配任务并自动生成 PR

核心功能

  • 四大入口 — CLI、IDE 扩展、桌面 App、Codex Cloud
  • AGENTS.md — 项目级 AI 指令手册，定义编码规范
  • Worktrees — Git 工作树隔离，并行执行多个任务
  • Skills — 将重复工作流封装为可复用技能
  • MCP 集成 — 连接 Jira、Linear 等外部工具
  • PR 自动化 — 在 GitHub Issue/PR 中 @codex 自动处理
  • 沙箱模式 — 安全审批机制，控制 Agent 权限

Cursor

基于 VS Code 的 AI 原生代码编辑器，将大语言模型深度集成到编辑体验的每一层。支持 Tab 补全、内联编辑、Agent 自主编程和 MCP 扩展。

快速入门

  1. 访问 cursor.com/download 下载安装包
  2. 首次启动时导入 VS Code 扩展和设置
  3. 打开项目文件夹：File → Open Folder
  4. 按 Cmd+I（Mac）或 Ctrl+I（Win）打开 Agent 面板
  5. 用 @ 引用文件，给 Agent 更多上下文
  6. 在 .cursor/rules/ 中创建项目规则文件

核心功能

  • Tab 补全 — 预测式 AI 自动补全，支持多行代码
  • Cmd+K — 选中代码后内联 AI 编辑
  • Agent 模式 — 自主搜索代码库、编辑文件、运行终端
  • Plan 模式 — Shift+Tab 先制定计划再执行
  • Rules — .cursor/rules/ 定义项目编码规范
  • MCP 服务器 — 连接数据库、GitHub 等外部系统
  • Cloud Agents — 浏览器/手机远程启动编程任务

快捷键功能
