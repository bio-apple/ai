AI 学习路线

从零基础到进阶应用，按阶段掌握主流 AI 工具与实战技能。

第 1 周

建立 AI 基础认知

ChatGPT 豆包 DeepSeek
  • 注册并体验 2 个对话 AI
  • 学会写清晰的提示词
  • 完成 3 个日常任务（邮件/总结/翻译）
第 2-3 周

掌握长文档与深度分析

Kimi Claude 通义千问
  • 上传 PDF 并生成摘要
  • 用 AI 分析合同或论文
  • 建立个人知识库（Projects）
第 4-6 周

编程与自动化

Cursor Microsoft Copilot OpenAI Codex
  • 配置 Cursor Rules
  • 用 Agent 完成一个小功能
  • 尝试 Codex PR 自动化
持续进阶

多工具协同工作流

Google Gemini 豆包
  • 按场景选择最优工具组合
  • 保存并复用高效提示词
  • 跟进本站每日视频与案例

学习路径速览

零基础入门

  1. 从 ChatGPT、豆包或 DeepSeek 开始，学会基本对话和提示词
  2. 用 Kimi 或通义千问处理长文档与中文场景任务
  3. 尝试 Gemini 的文件上传和 Deep Research 功能
  4. 用 Claude 处理复杂分析和高质量写作

进阶应用

  1. 编程方向：安装 Cursor 或 GitHub Copilot，配置 Rules 工作流
  2. 自动化方向：用 Codex 连接 GitHub 实现 PR 自动化
  3. 国内开发：用通义 API 或 DeepSeek API 低成本接入产品
  4. 无代码方向：用扣子 Coze 基于豆包搭建 AI 工作流
